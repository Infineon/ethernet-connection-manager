typedef cy_rslt_t(* cy_ecm_phy_init) (uint8_t eth_idx, ETH_Type *reg_base)

ECM PHY Init callback function pointer type.

Callback functions are invoked when the ECM needs to call PHY Ethernet driver API's during driver initialization. Note: The callback function will be executed in the context of the ECM.

◆ cy_ecm_event_callback_t

typedef void(* cy_ecm_event_callback_t) (cy_ecm_event_t event, cy_ecm_event_data_t *event_data)

ECM event callback function pointer type; events are invoked when the Ethernet driver posts events to ECM.

Parameters
[in]event: ECM events
[in]event_data: A pointer to the event data. The event data will be freed once the callback returns from the application.

Note: The callback function will be executed in the context of the ECM.
